## Git Installation

### 1. [Git Install Windows ver.](https://git-scm.com/install/windows)

### 2. Git Install MacOS ver.

- **Homebrew 설치** : homebrew는 macOS 운영 체제의 소프트웨어 패키지 관리 시스템이다.

&nbsp;   1. Terminal 열기

&nbsp;   2. 아래 명령어 실행 

&nbsp;   ```/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"```

- **Git 설치**

&nbsp;   1. Terminal에서 아래 명령어 실행

&nbsp;   ```brew install git```
